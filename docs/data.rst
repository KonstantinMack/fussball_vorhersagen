Data Sources:
-------------

* `SportDeer API <http://www.sportdeer.com>`__
* `football-data.co.uk <http://football-data.co.uk>`__
* `Transfermarkt <http://www.transfermarkt.de>`__


Data Structure:
---------------

- League
- Matchday
- Date
- Teams
- LineUps
- Match statistics: possesion, shots, shots on target, corners, cards
- Odds
- Result
